Personal website template

A lightweight, responsive personal website template built with plain HTML, CSS and JavaScript — free to use and customise for anyone who likes the design.

Introduction

This is a simple, original personal website template made with vanilla HTML, CSS and JavaScript. I created it when I first started learning UI/UX and HTML/CSS. It’s intentionally lightweight and approachable — perfect if you want a personal landing page without the overhead of frameworks. The website is fully responsive, designed with a mobile-first approach to ensure it looks and works great on all screen sizes — from smartphones to large desktop displays.

How to Use This Template

For a detailed step-by-step guide on how to publish your website using GitHub Pages, check out the official documentation: https://docs.github.com/en/pages/quickstart
